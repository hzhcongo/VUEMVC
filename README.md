# VUEMVC

## How to run the tests
1. Download IntelliJ project
2. Open IntelliJ project on IntelliJ
3. Select "Build Project" under "Build" tab
4. Select "Run 'Feature: ManageToDo" under "Run" tab
5. See "Run" tab for test results
